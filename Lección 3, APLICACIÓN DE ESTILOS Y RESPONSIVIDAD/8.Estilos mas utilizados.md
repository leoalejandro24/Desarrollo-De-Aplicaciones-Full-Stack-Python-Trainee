FUENTES

font-family: Define el tipo de letra
font-size: Tamaño del texto
font-weight: Grosor
font-style: Estilo

TEXTO

color: Color de texto
text-align: Alineación del texto
text-decoration: Subrayado-Tachado-etc..
line-height: Espacio entre lineas

LISTAS

list-style-type: Tipo de viñeta (circulo, número)
list-style-position: Imagen como viñeta
list-style-color: Ubicación de la viñeta

FONDO

background-color: Color de fondo
background-image: Imagen de fondo
background-repeat: Cómo se repite la imagen
background-position: Ubicación de la imagen

CONTENIDO

width / height: Ancho y alto del elemento
line-height: Espacio entre lineas
overflow: Como se muestra el contenido que excede el area

PADDING Y MÁRGENES

padding: Espacio interno del contenido
margin: Espacio externo del contenido
margin-top / padding-right: Aplicación individual por lado

BORDE

border-width: Groso del borde
border-style: Tipo de borde (línea, puntos, etc..)
border-color: Color del borde
border-radius: Redondeo de esquinas
