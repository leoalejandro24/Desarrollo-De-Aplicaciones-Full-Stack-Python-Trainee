Selectores en CSS son las instrucciones que se utilizan para "seleccionar" los elementos HTML
a los que se aplicará un estilo especifico.

SELECTOR POR ID

Se utiliza para aplicar estilos a un unico elemento HTML, el ID "id="se define en el HTML. Y en CSS
se define con #"nombredeid"

Ejemplo:

<!--HTML-->
<p id="Importante">Este parrafo tiene un ID único.</p> 
<p> Este parrafo no tiene un ID único.</p>
<!--CSS-->
#Importante {
    color: red;
    text-decoration: underline;
}

SELECTOR POR CLASE

Aplica estilos a varios elemtentos HTML que comparten clases. El class "class=" se define en el HTML.
Y en CSS se define con ."nombredeclass"

Ejemplo:

<!--HTML-->
<p class="destacado">Este parrafo usa una clase.</p> 

<!--CSS-->
.destacado {
    color: blue;
    font-weight: bold;
}
