# Selector de Elemento

Selecciona todos los elementos HTML del tipo especificado

Ejemplo: p <- Selecciona <p>

# Selector de identificación (ID)

Selecciona el elemento HTML con el ID especificado, solo se permite un unico elemento por ID

Ejemplo: #mi-id <- Selecciona <p id="mi-id"> y <a id="mi-id">

# Selector de clase

Selecciona los elementos en la pagina con la clase especificada

Ejemplo: .mi-clase <- Selecciona <p class="mi-clase"> y <a class="mi-clase">

# Selector de atributo

Selecciona los elementos en una pagina con el atributo especificada

Ejemplo: img[src] <- Selecciona <img src="miimagen.png"> pero no <img>

# Selector de pseudoclase

Selecciona los elementos especificado pero solo cuando este en el estado especificado,
por ejemplo cuando el puntero esta sobre el.

Ejemplo: a:hover <- Selecciona <a>, pero solo cuando el puntero está sobre el enlace

# Dar Estilo al Cuerpo de un Documento

body {
    width: 600px; <Esto hará que el cuerpo siempre tenga 600 píxeles de ancho.>
    margin: 0 auto; <Cuando seleccionamos dos valores dentro de margin o padding. El primer
    valor afecta a el lado superior (top) e inferior (bottom). El segundo valor lado
    izquierdo (left) y derecho (right)>
    background-color: #FF9500; <Color de fondo del elemento>
    padding: 0 20px 20px 20px; <Damos espacio alrededor del contenido (arriba: 0, derecha:20 pixeles, abajo: 20 pixeles, y al lado izquierdo: 20 pixeles.)>
    border: 5px solid black; <Borde de 5 pixeles de ancho, continuo y de color negro alrededor del elemento body>
}

# Posicionar y dar Estilo al Título Principal de la Página

h1 {
    margin: 0; <Eliminamos los margenes de arriba y abajo>
    padding: 20px 0; <Un padding de 20 pixeles arriba y 0 hacia la derecha>
    color: #00539F; <Color del texto>
    text-shadow: 3px 3px 1px black; <Aplica una sombra al elemento:
    1.- El primer valor en píxeles asigna el desplazamiento horizontal de la sombra desde el texto, qué tan lejos la mueve a la derecha. Un valor negativo la moverá a la izquierda.
    2.- El segundo valor en píxeles asigna el desplazamiento vertical de la sombra desde el texto, qué tan lejos la mueve hacia abajo. En este ejemplo, un valor negativo la desplazaría hacia arriba.
    3.- El tercer valor en píxeles asigna radio de desenfoque de la sombra, un valor grande es igual a una sombra borrosa.
    4.- El cuarto valor asigna el color base de la sombra.>
}

# Centrar Imagen

img {
    display: block;
    margin: 0 auto;
}
<Finalmente, centra la imagen para hacer que luzca mejor. Puedes usar nuevamente el truco de margin:0 auto que usaste antes para body, pero existen diferencias que requieren que hagas algo más para que el código CSS funcione.
El elemento "body" es un elemento en nivel de bloque (block-level), lo que significa que tomará espacio en la página y que puede tener otros valores de espacio aplicables como margen. Las imágenes, por otra parte, son elementos inline, lo que quiere decir que no puedes aplicarles márgenes, debes dar a la imagen un comportamiento de block-level usando display:block;>
