# Front end 

Tiene relación con la parte frontal de una pagina web, ya sean colores, formas, efectos
y la estructura de la misma.

# Back end

Es la capa de programación, es la que maneja los datos y los procesa para ser enviados al Front-end
y visibles por el usuario, que tan robusta sea esta parte nos dira que tan robusto es el sitio
en general

# Full Stack
